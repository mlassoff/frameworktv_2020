(Covid-19) Code Camp Day 1

Stuck Inside? Learn New Skills While Social Distancing

Don't forget to sign up for the full Covid-19 Code Camp to get access to all resources and materials.

The Covid-19 Code Camp is a specal opportunity for you to learn while sequestered. Members of this free code camp will receive new activities each week day, until the end of the Covid-19 sequester.

In Day 1 of camp, we start learning the Python programming language. Python is a powerful language, but easy to learn for beginners. For many this will be the first time they've worked in a shell and used a Linux-style command line to write and run code. Be sure to download the PDF code companion below.
